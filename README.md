# Custom Vite Template

```sh
# 저장소 복사
npx degit https://github.com/Hanna-Jeanne/likelion-homework-React <폴더이름>
```

```sh
# 해당 폴더 이동
cd <폴더이름>
```

```sh
# 패키지 설치
pnpm install
```

## 실행 명령어

```sh
# 개발 서버 구동
pnpm dev
```

```sh
# 프로젝트 빌드
pnpm build
```

```sh
# 프로젝트 프리뷰
pnpm preview
```

```sh
# 린팅 검사
pnpm lint
```

```sh
# 포멧팅
pnpm format
```
